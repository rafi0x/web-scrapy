## WEB SCRAPER

### @main features: 
- Scrape data from html page.
- Can get data for multiple pages.
- Export data as excel.
- Save data to google spreadsheet.

### How to run: 

    yarn install
    yarn start
  open index.html in browser from the public dir and ready to rock
